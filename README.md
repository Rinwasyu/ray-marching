# ray-marching

Ray marching program written in C.

## Requirements

- GCC
- OpenMP
- Image viewer for netpbm format

## Usage

Run below
```console
$ git clone https://github.com/Rinwasyu/ray-marching.git
$ cd ray-marching/src
$ make
$ ./main
```

then open `result.ppm`.
